V UNISEX SALON APP UI/UX DESIGN

Project Overview
----------------
V Unisex Salon App is a mobile application UI/UX design created in Figma
for salon appointment booking and service management.

The application allows users to:
- Login and Register
- Browse salon services
- Search for services
- View service pricing
- Book appointments
- Select available time slots
- Receive booking confirmation

Features
--------
1. Splash Screen
2. Login Screen
3. Home Screen
4. Search Screen
5. Haircut Details
6. Massage Details
7. Facial Details
8. Hair Colour Details
9. Appointment Booking
10. Booking Confirmation
